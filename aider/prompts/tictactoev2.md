Instructions: Create a Python terminal game for two-player tic-tac-toe. The application should be offline and require no internet access. Players alternate playing on the same computer, taking turns as X and O.

Context: Tic Tac Toe is a two-player game played on a 3x3 grid. Players alternate placing their mark (X or O) in empty squares, with the goal of getting three in a row horizontally, vertically, or diagonally. The first player to do so wins, while a full board with no winner results in a draw.

Input Data: Players select squares by entering a number 1-9. Validate input by rejecting occupied squares and non-valid entries with a clear message, then re-prompt the player.

Output Indicator: Display the board after each move with a numbered reference grid. Show whose turn it is before each move. Announce the winner or declare a draw when the game ends. Offer a "play again?" option after each game.
